갤럭시 반납 초기화 콘솔

세린캠 대여폰 정리 · WebUSB + ADB

상태 미연결

1 · 기기 연결

USB로 연결된 갤럭시를 선택하세요.
내부 저장공간—

2 · 정리 항목 선택

유지되는 항목: 삼성 계정 로그인 · 카메라 설정 · 설치된 앱 자체는 그대로 둡니다. 아래 작업은 삼성 계정을 절대 로그아웃하지 않습니다.

3 · 실행

처음 사용 전 준비

이 페이지는 PC의 Chrome / Edge에서 열어야 하며, HTTPS(GitHub Pages 등) 또는 localhost에서만 동작합니다.
  1. 대여폰: 설정 → 휴대전화 정보 → 소프트웨어 정보 → 빌드번호 7번 탭 → 개발자 옵션 활성화
  2. 개발자 옵션 → USB 디버깅 켜기
  3. USB 케이블로 PC 연결 → 폰에 뜨는 USB 디버깅 허용 팝업에서 "항상 허용" 체크 후 허용
  4. PC에서 다른 ADB(Android Studio 등)가 실행 중이면 adb kill-server로 종료
삼성 계정·카메라 설정은 건드리지 않으므로, 재세팅 없이 다음 고객에게 바로 대여할 수 있습니다.
